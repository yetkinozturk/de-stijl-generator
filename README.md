# De Stijl Generator


https://yetkinozturk.github.io/de-stijl-generator/

## Requirements

- **Java 11+** (Shadow-CLJS runs on the JVM)
- **Bun** — https://bun.sh

## Setup

```bash
# Install JS dependencies with Bun
bun install

# Start dev server with hot reload at http://localhost:3000
bun run dev

# Production build → public/
bun run build
